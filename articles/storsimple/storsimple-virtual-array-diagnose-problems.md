# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Řešení potíží se službou StorSimple Virtual Array s využitím služby Správce zařízení StorSimple
## <a name="overview"></a>Přehled

StorSimple Device Manager poskytuje nastavení **diagnostiky a řešení problémů** v okně s přehledem služby, které zvýrazňuje některé běžné problémy, ke kterým může docházet ve virtuálním poli a jak je řešit. V tomto kurzu se seznámíte s funkcemi pro řešení potíží samoobslužné služby, které poskytuje služba StorSimple Device Manager.

![Diagnostikovat a řešit problémy](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostikovat a řešit problémy

Můžete si prohlédnout některé běžné problémy s virtuálním polem StorSimple a prohlédnout si řešení těchto problémů přímo v okně s přehledem služby StorSimple Device Manager.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Diagnostika problému s virtuálním polem

1. Pokud chcete zobrazit seznam běžných problémů, klikněte v levém podokně okna StorSimple Device Manager služby na možnost **Diagnostika a řešení problémů** .

2. **Rozbalte** příznak problému, ke kterému se pokoušíte zkontrolovat **Doporučené kroky** , abyste vám pomohli při řešení tohoto problému. Pokud chcete, můžete si také projít podrobné odkazy na dokumentaci, která je k dispozici pro další informace.
   
    ![Řešení problému](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Pokud nemůžete najít odkaz na váš problém nebo ho vyřešit, požádejte o další pomoc podpora Microsoftu.
