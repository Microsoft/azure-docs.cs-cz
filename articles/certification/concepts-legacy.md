# <a name="legacy-devices-on-the-azure-certified-device-catalog"></a>Starší zařízení v katalogu zařízení s certifikací Azure

Možná jste si všimli v katalogu zařízení s certifikací Azure, že některá zařízení nemají modrý obrys nebo popisek zařízení s certifikací Azure. Tato zařízení (dubbed "starší zařízení") byla dříve certifikována v rámci starší verze programu.

## <a name="certified-for-azure-iot-program"></a>Program Certified for Azure IoT

Před spuštěním programu zařízení s certifikací Azure by mohli hardwaroví partneři dřív certifikovat své produkty v rámci programu Certified for Azure IoT. Program certifikace zařízení s certifikací Azure se zaměřuje na svou misi, aby poskytoval příslibů zákazníkům, a ne k technickým funkcím zařízení.

Zařízení, která jsou certifikována jako zařízení s certifikací IoT Hub, se v katalogu zařízení s certifikací Azure zobrazí jako starší zařízení. Tento popisek označuje zařízení, která byla dříve kvalifikována prostřednictvím již vyřazeného programu, ale nebyla certifikována prostřednictvím aktualizovaného programu zařízení s certifikací Azure. Tato zařízení jsou jasně popsána v katalogu podle jejich nedostatku modrého přehledu a lze je najít prostřednictvím filtru "IoT Hub certifikováno zařízení (starší verze)".
