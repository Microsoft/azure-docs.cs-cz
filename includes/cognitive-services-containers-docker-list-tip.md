> [!TIP]
> K vypsání stažených imagí kontejneru můžete použít příkaz [Docker images](https://docs.docker.com/engine/reference/commandline/images/) . Například následující příkaz vypíše ID, úložiště a značku jednotlivých stažených imagí kontejneru, které jsou formátovány jako tabulka:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
