## <a name="sign-in-to-azure"></a>Přihlášení k Azure

Než můžete aplikaci publikovat, musíte se přihlásit do Azure.

1. Pokud ještě nejste přihlášení, zvolte na řádku aktivity ikonu Azure a pak v oblasti **Azure: funkce** zvolte **Přihlásit se k Azure...**. Pokud ho ještě nemáte, můžete si **[vytvořit bezplatný účet Azure](https://azure.microsoft.com/free/)**. Studenti můžou **[vytvořit bezplatný účet Azure pro studenty](https://azure.microsoft.com/free/students/)**.

    ![Přihlaste se k Azure v rámci VS Code](./media/functions-sign-in-vs-code/functions-sign-into-azure.png)

    Pokud jste už přihlášení, přečtěte si další část.

1. Po zobrazení výzvy v prohlížeči vyberte svůj účet Azure a přihlaste se pomocí svých přihlašovacích údajů k účtu Azure.

1. Po úspěšném přihlášení můžete zavřít nové okno prohlížeče. Předplatná, která patří k vašemu účtu Azure, se zobrazí na bočním panelu.
