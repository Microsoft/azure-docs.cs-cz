# <a name="azure-machine-learning-curated-environments"></a>Azure Machine Learning v nich uspořádaná prostředí

Tento článek obsahuje seznam zobrazených prostředí v Azure Machine Learning. Dodaná prostředí jsou poskytována Azure Machine Learning a jsou ve výchozím nastavení k dispozici ve vašem pracovním prostoru. Jsou zajištěny pomocí imagí Docker uložených v mezipaměti, které používají nejnovější verzi sady Azure Machine Learning SDK, snížení nákladů na přípravu na běh a umožnění rychlejšího nasazení. Pomocí těchto prostředí můžete rychle začít s různými architekturami strojového učení.

> [!NOTE]
> Tento seznam se aktualizuje od prosince 2020. Pomocí sady Python SDK získáte nejaktuálnější seznam prostředí a jejich závislosti. Další informace najdete v článku o [prostředích](./how-to-use-environments.md#use-a-curated-environment).
