# <a name="quickstart-create-an-open-source-redis-cache"></a>Rychlý Start: vytvoření open source mezipaměti Redis Cache

Azure cache for Redis poskytuje plně spravované [Open Source Redis](https://redis.io/) v rámci Azure. Můžete začít s mezipamětí Azure cache pro instanci Redis libovolné vrstvy (Basic, Standard nebo Premium) a velikostí a škálovat ji tak, aby splňovala požadavky na výkon vaší aplikace. Tento rychlý Start ukazuje, jak použít Azure Portal k vytvoření nové mezipaměti Azure pro Redis.

## <a name="prerequisites"></a>Požadavky

Než začnete, budete potřebovat předplatné Azure. Pokud ho ještě nemáte, vytvořte nejdřív [bezplatný účet](https://azure.microsoft.com/free/) .
