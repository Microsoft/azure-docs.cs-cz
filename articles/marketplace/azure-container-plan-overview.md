# <a name="create-and-edit-plans-for-an-azure-container-offer"></a>Vytvoření a úprava plánů nabídky Azure Container

Tato stránka přehledu vám umožní vytvořit různé možnosti plánu v rámci stejné nabídky. Plány (dříve nazývané SKU) se můžou lišit v tom, kde jsou dostupné (globální Azure nebo Azure Government), a image, na kterou plán odkazuje. Vaše nabídka musí obsahovat aspoň jeden plán.

Pro každou nabídku můžete vytvořit až 100 plánů: až 45 z nich může být privátní. Další informace o privátních plánech najdete v [části soukromé nabídky na komerčním webu Microsoft Marketplace](private-offers.md).

Po vytvoření plánu se zobrazí stránka **Přehled plánu** :

- Názvy plánů
- Cenový model
- Oblasti Azure (globální nebo státní správa)
- Stav aktuálního publikování
- Jakékoli dostupné akce

Akce dostupné pro plán se liší v závislosti na aktuálním stavu plánu. Mezi ně patří:

- **Odstraní koncept** , pokud je stav plánu koncept.
- Pokud je stav plánu publikovaný živě, **zastaví se prodej plánu** .

## <a name="edit-a-plan"></a>Upravit plán

Vyberte **název** plánu a upravte jeho podrobnosti.

## <a name="create-a-plan"></a>Vytvoření plánu

Pokud chcete nastavit nový plán, vyberte **+ vytvořit nový plán**.

Zadejte jedinečné **ID plánu** pro každý plán. Toto ID bude viditelné pro zákazníky na webové adrese daného produktu. Používejte jenom malá písmena a číslice, pomlčky nebo podtržítka a maximálně 50 znaků. Po zvolení **Create (vytvořit**) nemůžete změnit ID plánu.

Zadejte **název plánu**. Zákazníci uvidí tento název při rozhodování, který plán vybrat v rámci vaší nabídky. Každý plán v této nabídce musí mít jedinečný název. Můžete například použít název nabídky **Windows serveru** s plány **Windows Server 2016** a **Windows Server 2019**.

Vyberte **vytvořit** a pokračovat níže.
