Při práci s modelem nasazení Classic nemůžete použít Azure Cloud Shell. Místo toho musíte na svém počítači nainstalovat nejnovější verzi rutin PowerShellu pro správu služeb Azure (SM). Tyto rutiny se liší od rutin AzureRM nebo AZ. Pokud chcete nainstalovat rutiny SM, přečtěte si téma [instalace rutin služby Service Management](/powershell/azure/servicemanagement/install-azure-ps). Další informace o Azure PowerShell obecně najdete v [dokumentaci k Azure PowerShell](/powershell/azure/).
