# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Začínáme s vývojem zařízení Azure IoT Embedded

**Platí pro**: [vývoj integrovaných zařízení](about-iot-develop.md#embedded-device-development)

Tato příručka Začínáme obsahuje sadu rychlých startů, které vám ukáže, jak začít pracovat s integrovanými zařízeními a Azure IoT. 

V každém rychlém startu dokončíte následující základní úlohy:
* Instalace sady integrovaných vývojářských nástrojů pro programování určitého zařízení v jazyce C
* Sestavte image, která zahrnuje komponenty a ukázky Azure RTO, a potom zařízení flash
* Zabezpečené připojení zařízení k Azure IoT
* Zobrazení telemetrie zařízení, zobrazení vlastností a volání metod Cloud-zařízení
