# <a name="texconv---texture-conversion-tool"></a>Nástroj pro převod textury TexConv

Nástroj TexConv je externí nástroj příkazového řádku, který umožňuje provádět převody z dávkových textur.
Tento nástroj najdete v tomto úložišti Git: [úložiště GitHub pro nástroj TexConv](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Pro vzdálené vykreslování nemusí být textury převáděny, protože krok převodu se může zabývat běžnými formáty souborů obrázků a převést textury na příslušný formát modulu runtime. Tento nástroj ale může být užitečný k převedení textur místně s vlastním nastavením.
