# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Zobrazit prostředek datového kontroleru Azure ARC v Azure Portal

Jakmile provedete první [metriky nebo protokoly nahrávání do Azure](upload-metrics-and-logs-to-azure-monitor.md) nebo [nahráli využití](view-billing-data-in-azure.md), uvidíte, že prostředek řadiče dat ARC Azure, který je vytvořený v Azure, v Azure Portal.

Otevřete Azure Portal pomocí speciální adresy URL:  [https://portal.azure.com](https://portal.azure.com) .

Na panelu hledání vyhledejte svůj řadič dat podle názvu a klikněte na něj.
