# <a name="overview-of-the-media-blueprint-sample"></a>Přehled ukázky podrobného plánu pro média

Ukázka podrobného plánu pro média poskytuje sadu ochranných mantinelů zásad správného řízení s využitím [Azure Policy](../../../policy/overview.md), které pomáhají zajistit osvědčení pro [média](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Ukázka podrobného plánu

Ukázka podrobného plánu pomáhá zákazníkům nasadit základní sadu zásad pro libovolnou architekturu nasazenou v Azure, která vyžaduje osvědčení pro architekturu médií nebo dodržování příslušných předpisů. Oddíl [mapování kontrol](./control-mapping.md) poskytuje podrobné informace o zásadách zahrnutých v této iniciativě a o tom, jak tyto zásady pomáhají splňovat různé kontroly definované architekturou médií. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje dodržování přiřazených zásad.
