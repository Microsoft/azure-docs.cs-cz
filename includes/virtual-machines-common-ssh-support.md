## <a name="supported-ssh-key-formats"></a>Podporované formáty klíče SSH

Azure v současné době podporuje páry klíčů veřejného a soukromého klíče RSA protokolu SSH (2) s minimální délkou 2048 bitů. Jiné formáty klíčů jako ED25519 a ECDSA se nepodporují.
