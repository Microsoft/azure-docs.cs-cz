## <a name="create-and-manage-azure-sql-resources-with-the-azure-portal"></a>Vytváření a Správa prostředků Azure SQL pomocí Azure Portal

Azure Portal poskytuje jednu stránku, kde můžete spravovat [všechny prostředky Azure SQL](https://go.microsoft.com/fwlink/?linkid=2100641) , včetně vašich virtuálních počítačů SQL.

Pokud chcete získat přístup ke stránce **Azure SQL** , v nabídce Azure Portal vyberte **Azure SQL** nebo vyhledejte a na libovolné stránce vyberte **Azure SQL** .

> [!NOTE]
> **Azure SQL** nabízí rychlý a snadný způsob přístupu ke všem prostředkům SQL v Azure Portal, včetně jedné a sdružené databáze v Azure SQL Database, i logických SQL Server, které jsou hostované, SQL spravované instance a virtuální počítače SQL.  [Azure SQL](../azure-sql-iaas-vs-paas-what-is-overview.md) není služba ani prostředek, ale spíše rodina služeb souvisejících s SQL. 

Pokud chcete spravovat existující prostředky, vyberte požadovanou položku v seznamu. Pokud chcete vytvořit nové prostředky Azure SQL, vyberte **+ Přidat**. 

![Stránka portálu Azure SQL](./media/sql-database-create-manage-portal/add-azure-sql-resources.png)

Po výběru možnosti **+ Přidat**zobrazte další informace o různých možnostech výběrem možnosti **Zobrazit podrobnosti** na libovolné dlaždici.

![Podrobnosti dlaždice databáze](./media/sql-database-create-manage-portal/single-sql-database-deployment-options.png)
