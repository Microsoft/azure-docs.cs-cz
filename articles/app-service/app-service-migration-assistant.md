# <a name="migrate-to-azure-app-service"></a>Migrace na Azure App Service

Pomocí [App Service Pomocník s migrací](https://azure.microsoft.com/services/app-service/migration-assistant/) můžete migrovat místní aplikaci do Azure App Service. App Service Pomocník s migrací je navržená tak, aby zjednodušila svoji cestu do cloudu prostřednictvím bezplatného, jednoduchého a rychlého řešení pro migraci aplikací z místního prostředí do cloudu.

Pomocí Azure App Service Pomocník s migrací můžete rychle:

- Zkontrolujte adresu URL aplikace a vyhodnoťte, jestli je vhodným kandidátem na migraci.
- Stáhněte si Pomocník s migrací pro zahájení migrace.
- Použijte nástroj ke spuštění kontrol připravenosti a obecného vyhodnocení nastavení konfigurace vaší aplikace.
- Migrujte svou aplikaci nebo web a Azure App Service pomocí nástroje.

[Podívejte se, jak migrovat webové aplikace do služby Azure App Service](https://www.youtube.com/watch?v=9LBUmkUhmXU).

Další krok:  [migrace místní webové aplikace na Azure App Service](https://docs.microsoft.com/learn/modules/migrate-app-service-migration-assistant/)
