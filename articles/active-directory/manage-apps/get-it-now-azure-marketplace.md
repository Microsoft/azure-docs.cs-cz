# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Získat hned přidat aplikaci z Azure Marketplace

Už to skoro je! 

Pokud se pokoušíte použít službu Azure AD jako zprostředkovatele identity pro aplikaci, jste na správném místě. Stačí ho přidat do svého tenanta Azure AD. Další informace o tom, jak to udělat, najdete v části rychlý Start tady: [zobrazení aplikací ve vašem Tenantovi Azure AD](view-applications-portal.md).

Pokud se pokoušíte nainstalovat aplikaci do místního počítače nebo mobilního zařízení, budete mít na špatném místě. Azure Marketplace je určený pro organizace, které používají aplikace se svými klienty Azure. V případě osobních počítačů a zařízení se podívejte na [obchod s aplikacemi Microsoftu](https://www.microsoft.com/store/apps).
