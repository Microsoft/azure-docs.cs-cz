# <a name="azure-storage-analytics-and-big-data-partners"></a>Partneři Analýza úložiště Azure a velkých objemů dat

Tento článek popisuje partnerské společnosti Microsoftu, které jsou integrované s Azure Data Lake Storage. Zahrnují úlohy, včetně moderních datových skladů, pokročilých analýz, analýz v reálném čase atd. Tito partneři využívají hierarchický obor názvů v Azure Storage k optimalizaci řešení a jeho efektivnímu spuštění v Azure.
