> [!IMPORTANT]
> [Microsoft Identity Platform (v 2.0)](/azure/active-directory/develop/v2-overview) představuje [vývoj platformy pro vývojáře Azure Active Directory (Azure AD) (verze 1.0)](../articles/active-directory/azuread-dev/about-microsoft-identity-platform.md). Umožňuje vývojářům vytvářet aplikace, které přihlašují všechny identity od Microsoftu a získávají tokeny pro volání rozhraní API od Microsoftu, jako jsou Microsoft Graph nebo rozhraní API, která vytvořili vývojáři.
> Tento obsah je pro starší koncový bod Azure AD v 1.0. Pro nové projekty doporučujeme použít koncový bod v 2.0. Další informace najdete v tématu [Proč aktualizace pro Microsoft Identity Platform (v 2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) i [omezení platformy Microsoft Identity](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
