# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Zobrazení inventáře vašich instancí v Azure Portal

Až nahrajete [metriky, protokoly](upload-metrics-and-logs-to-azure-monitor.md)nebo [využití](view-billing-data-in-azure.md), můžete svou instanci zobrazit jako prostředek Azure z Azure Portal. Pokud chcete svůj prostředek zobrazit v Azure Portal, použijte speciální adresu URL [https://portal.azure.com](https://portal.azure.com) a proveďte tyto kroky:

1. Přejít na **všechny služby**.
1. Vyhledejte typ instance databáze.
1. Přidejte typ k oblíbeným položkám.
1. V nabídce vlevo vyberte typ instance.
1. Podívejte se na vaše instance ve stejném zobrazení jako vaše jiné prostředky Azure SQL nebo Azure PostgreSQL s škálovatelným škálováním (používejte filtry pro podrobné zobrazení).
