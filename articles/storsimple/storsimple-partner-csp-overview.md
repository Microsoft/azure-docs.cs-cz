# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>Co je StorSimple pro poskytovatele cloudových řešení?


## <a name="overview"></a>Přehled

Microsoft Azure StorSimple je jedinečný přístup k podnikovým úložištím s skutečnými možnostmi hybridního cloudového úložiště. Díky tomu můžou zákazníkům využít výhod ekonomického cloudového úložiště pro neaktivní data a uchovávat jejich nejdůležitější data v místním prostředí pro nejvyšší úroveň výkonu. 

StorSimple Virtual Array pro Cloud Solutions Provider (CSP) umožňuje partnerům na tuto příležitost využít velká písmena. Partneři vlastní životní cyklus zákazníka s přímým zřizováním, fakturací a podporou cloudových služeb Microsoftu. V krátké době si partneři teď můžou StorSimple spolu s jejich službami zákazníkům.

Další informace o StorSimple pro zprostředkovatele CSP najdete na stránce [Přehled CSP pro Azure](/azure/cloud-solution-provider/overview/azure-csp-overview) .

Další informace o fakturaci, cenách, pobídek a získání podpory v CSP najdete v tématu [StorSimple in CSP: nejčastější dotazy](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Nasazení a Správa StorSimple pro CSP

StorSimple pro CSP je k dispozici jako služba založená na využití na všech trzích, kde je StorSimple k dispozici dnes. StorSimple pro CSP používá Azure Portal a službu StorSimple Správce zařízení. Partner CSP může vytvořit StorSimple Správce zařízení pro správu virtuálních polí StorSimple, sdílených složek, svazků a záloh. Pomocí Azure Portal můžete spravovat všechna virtuální pole zaregistrovaná ve službě StorSimple Správce zařízení.

Další informace najdete v, jak [nasadit a spravovat virtuální pole StorSimple pro CSP](storsimple-partner-csp-deploy.md).
