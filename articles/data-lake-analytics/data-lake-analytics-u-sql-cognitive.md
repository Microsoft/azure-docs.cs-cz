# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Začínáme se schopností rozpoznávání U-SQL

## <a name="overview"></a>Přehled
Schopnosti vnímání u-SQL umožňují vývojářům používat do svých programů pro velké objemy dat funkce Put Intelligence. 

K dispozici jsou následující ukázky, které používají funkce rozpoznávání:
* Vytváření imagí: [detekce plošek](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Bitové kopie: [detekce emoce](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Bitové kopie: [rozpoznání objektů (označování)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Bitové kopie: [OCR (optické rozpoznávání znaků)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Text: [Extrakce klíčových frází & analýza mínění](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
