# <a name="get-started-with-the-onedrive-for-business-connector"></a>Začínáme s konektorem OneDrivu pro firmy
Připojte se k OneDrivu pro firmy a spravujte soubory. Můžete provádět různé akce, jako je například nahrávání, aktualizace, získání a odstranění souborů.

Můžete začít vytvořením aplikace logiky hned teď v tématu [Vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-onedrive-for-business"></a>Vytvoření připojení k OneDrivu pro firmy
Pokud chcete vytvořit Logic Apps s OneDrivem pro firmy, musíte nejdřív vytvořit **připojení** a pak zadat podrobnosti pro následující vlastnosti:

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| Token |Ano |Zadat přihlašovací údaje pro OneDrive pro firmy |

Jakmile vytvoříte připojení, můžete ho použít ke spuštění akcí a naslouchat aktivačním událostem popsaným v tomto článku.
