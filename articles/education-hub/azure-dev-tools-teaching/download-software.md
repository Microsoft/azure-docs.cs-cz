# <a name="help-students-download-software-developer-tools-from-the-azure-education-hub"></a>Pomáhat studentům stahovat nástroje pro vývojáře softwaru z Azure Centrum Education

Pomocí vývojářských nástrojů Azure pro vzdělávací program můžou studenti, kteří mají stávajícího a aktivní předplatné Azure for Students, stáhnout některé nástroje pro vývojáře softwaru zdarma. Azure for Students poskytuje přístup ke všem produktům Azure, které jsou výslovně určené pro podporu vzdělávání nebo výuky, nekomerčního výzkumu nebo úsilí o návrh, vývoj, testování a předvedení softwarových aplikací pro tyto účely. Studenti stáhnou nástroje z Azure Centrum Education.

## <a name="instructions-for-students-how-to-download-software-developer-tools"></a>Pokyny pro studenty: stažení nástrojů pro vývojáře softwaru

Pořiďte si své studenty pomocí tohoto postupu a Stáhněte si nástroje pro vývojáře softwaru:

1. Otevřete prohlížeč a přejdete na [centrum Education Azure](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).
1. Přihlaste se pomocí přihlašovacích údajů předplatného Azure for Students.
1. Vyberte kterýkoli z vývojářských nástrojů softwaru v Centrum Education otevřete podokno stahování.
1. Přečtěte si informace o softwaru a ujistěte se, že je software kompatibilní s vaším počítačovým systémem.
1. Po potvrzení, že je software kompatibilní s vaším systémem, vyberte **Stáhnout**.
1. Postupujte podle pokynů na obrazovce a stáhněte a nainstalujte software.

   Azure for Students vám poskytne $100 kredit na 12 měsíců. Zahrnuje přístup k více než 25 bezplatným službám, včetně výpočetních prostředků, sítě, úložiště a databází. Veškeré poplatky vzniklé během tohoto období se odečtou z kreditu.
