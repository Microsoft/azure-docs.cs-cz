# <a name="team-data-science-process-for-developer-operations"></a>Vědecké zpracování týmových dat pro vývojářské operace

Tento článek popisuje funkce vývojářské operace (DevOps), které jsou specifické pro pokročilé analýzy a implementaci Cognitive Services řešení. Tyto školicí materiály implementují TDSP (Team data vědu) a software a sady Microsoft a open source a sady nástrojů, které jsou užitečné pro vytváření a poskytování řešení pro datové vědy. Odkazuje na témata, která se týkají DevOps sada nástrojů, která je specifická pro datové vědy a projekty a řešení AI.
