# <a name="azure-security-overview"></a>Přehled zabezpečení Azure

Informace o zabezpečení Microsoft Azure web na Azure.com je místo pro technické odpovědi na své otázky zabezpečení Azure. Pokud vás zajímají dodržování předpisů Azure a jak zabezpečuje svou vlastní infrastrukturu Azure, přejděte [Microsoft Trust Center](https://microsoft.com/en-us/trustcenter/default.aspx).

Víme, že je zabezpečení stavbách v cloudu a jak důležité je, že zjistíte přesné a aktuální informace o zabezpečení Azure. Jedním z nejlepších důvody pro použití Azure pro vaše aplikace a služby je využít širokou škálu nástrojů zabezpečení a možnosti, které jsou k dispozici. Tyto nástroje a možnosti nápovědy umožňují vytvářet bezpečná řešení na zabezpečené platformě Azure.

Na této stránce a tento web bude vyvíjet nové služby Azure do režimu online a stávající služby se aktualizují. Neustále vylepšit naše nabídky zabezpečení a zveřejníme technické podrobnosti o těchto vylepšeních tady. Doporučujeme, abyste přejděte na tento web pravidelně a podívejte se, co je nového!

## <a name="using-the-azure-security-information-site"></a>Použití webu s informacemi o zabezpečení Azure

Když se podíváte na levé straně stránky se zobrazí kolekce kategorií témat týkajících se zabezpečení. Klikněte nebo klepněte na téma najdete v článcích, který máme k dispozici pro dané kategorie.

Každá kategorie má přehledovém článku. Cílem přehledovém článku je poskytnout přehled co Azure může nabídnout z hlediska zabezpečení pro tuto kategorii. Přehledové články vám pomůžou pochopit produktech, službách a funkcích dostupných v Azure, které můžete použít ke zvýšení zabezpečení všech vašich nasazení. Tyto články přehled obsahují řadou odkazů, které technický obsah tak, aby další informace o tématech a pro jejich implementaci.

Webu s informacemi o zabezpečení Azure aktuálně řeší následující kategorie:

* Zdroje a prostředky
* Zabezpečení sítě
* Zabezpečení úložiště
* Zabezpečení virtuálních počítačů
* Ochrana koncových bodů
* Monitorování a správa zabezpečení
* Správa identit
* Služby PaaS
* Konkrétní odvětví
* Architektura

> [!NOTE]
> Všechny články, které máme v lokalitě informace o zabezpečení Azure se zobrazí v navigačním panelu na levé straně stránky. Pokud chcete zobrazit všechny články, klikněte nebo klepněte na **zobrazit další** v dolní části seznamu kategorií.
