# <a name="microsoft-azure-documentation"></a>Dokumentace k Microsoft Azure

Vítá vás opensourcová [dokumentace](https://docs.microsoft.com/azure) pro [Microsoft Azure](https://azure.microsoft.com). Přečtěte si prosím tento soubor README a seznamte se s tím, jak můžete přispívat do dokumentace k Microsoft Azure.

## <a name="getting-started"></a>Začínáme

Přispívání v rámci opensourcu neznamená jenom poskytovat aktualizace, ale také dát vědět v případě potíží. Další informace najdete v tématu s [pokyny pro přispívání](.github/CONTRIBUTING.md).

### <a name="prerequisites"></a>Požadavky

Rozhodli jste se přispívat, to je skvělé! Abyste mohli přispívat do dokumentace, potřebujete několik nástrojů.

Přispívání do dokumentace vyžaduje účet GitHub. Pokud tento účet nemáte, postupujte podle pokynů pro [nastavení účtu GitHub](https://docs.microsoft.com/contribute/get-started-setup-github) v naší příručce pro přispěvatele.

#### <a name="download"></a>Ke stažení

Nainstalujte si následující nástroje:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* Rozšíření [Docs Authoring Pack](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) pro Visual Studio Code

#### <a name="install"></a>Instalace

Postupujte podle pokynů uvedených v tématu věnovaném [instalaci nástrojů pro vytváření obsahu](https://docs.microsoft.com/contribute/get-started-setup-tools) v naším příručce pro přispěvatele.
