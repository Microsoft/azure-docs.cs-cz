Azure Cloud Shell je prostředí založené na prohlížeči, které je určené ke správě a vývoji prostředků Azure.

Cloud Shell nabízí předem nakonfigurovaná prostředí prostředí pro správu prostředků Azure bez režie instalace, správy verzí a údržby počítače.

Cloud Shell zřídí počítače na základě jednotlivých požadavků a v důsledku toho nebude stav počítače v relacích zachován. Vzhledem k tomu, že je Cloud Shell sestaven pro interaktivní relace, prostředí se automaticky ukončí po 20 minutách nečinnosti prostředí.
