# <a name="securing-external-collaboration-in-azure-active-directory-and-microsoft-365"></a>Zabezpečení externí spolupráce v Azure Active Directory a Microsoft 365

Zabezpečená spolupráce s externími partnery zajišťuje, aby měli dopraví další externí partneři odpovídající přístup k interním prostředkům po správnou dobu. Prostřednictvím přístupu pomocí zásad správného řízení holistický můžete snížit rizika zabezpečení, splnit cíle dodržování předpisů a zajistit, že víte, kdo má přístup.

Neupravená spolupráce vede k nedostatku srozumitelnosti o vlastnictví přístupu a možnosti zpřístupnění citlivých prostředků. Přechod na zabezpečenou a řízenou spolupráci zajistí, že existují jasné řádky vlastnictví a odpovědnosti pro přístup externích uživatelů. Sem patří:

* Správa externích organizací a uživatelů v nich, kteří mají přístup k prostředkům.

* Zajistěte, aby byl přístup vhodný, kontrolovaný a s časovým rozsahem v takovém případě.

* Umožňuje vlastníkům podniku spravovat spolupráci v rámci ochranných kolejnicí vytvořených IT.

Pokud musíte splnit architektury dodržování předpisů, můžete řídit spolupráci, která umožňuje ověřit správnou možnost přístupu.

Microsoft nabízí komplexní sady nástrojů pro zabezpečení externího přístupu.  Spolupráce B2B ve službě Azure Active Directory (Azure AD) je v centru všech externích plánů spolupráce. Azure AD B2B se dá integrovat s jinými nástroji v Azure AD a nástroji ve službě Microsoft 365 Services, aby bylo možné lépe zabezpečit a spravovat váš externí přístup.

Tato sada dokumentů je navržená tak, aby vám umožnila přejít ze služby ad hoc na bezpečnější stav a jak se jim bude volně vztahovat externí spolupráce.
