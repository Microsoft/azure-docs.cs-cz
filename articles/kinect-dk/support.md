# <a name="azure-kinect-support-options-and-resources"></a>Možnosti a prostředky podpory Azure Kinect

Tento článek vás provede různými možnostmi podpory.

## <a name="community-support"></a>Podpora komunity

Máte několik způsobů, jak získat odpovědi na vaše otázky prostřednictvím veřejných fór:

- [StackOverflow](https://stackoverflow.com/search?q=azurekinect&s=3b855ed0-8564-4961-856f-9614aeab4c0d&s=fd9ea920-622c-4d8e-b908-ec996e1f1403), kde můžete klást otázky nebo Hledat v existující knihovně otázek.
- [GitHub](https://github.com/Microsoft/Azure-Kinect-Sensor-SDK), kde můžete klást otázky, otevírat nové chyby nebo přispívat k vývoji sady Azure Kinect senzor SDK.
- [Poskytněte zpětnou vazbu](https://feedback.azure.com/forums/920053-azure-kinect-dk), kde můžete sdílet své nápady pro budoucnost produktu a hlasovat pro stávající nápad.

## <a name="assisted-support"></a>Podpora s asistencí

Existuje několik způsobů, jak získat podporu pro Azure Kinect s asistencí.

### <a name="microsoft-qa"></a>Microsoft Q&A

Pro zajištění rychlých a spolehlivých odpovědí na dotazy k technickým produktům od techniků Microsoftu, z Azure MVP (MVP) nebo z naší odborné komunity se můžete zapojit do [Microsoft Q&a](/answers/products/azure), preferovaný cíl Azure pro podporu komunity.

- [Microsoft Q&a pro Azure Kinect](/answers/topics/azure-kinect-dk.html), kde můžete klást otázky nebo Hledat v existující knihovně otázek.

### <a name="development-azure-kinect-on-azure"></a>Vývoj Azure Kinect v Azure

Předplatitelé Azure můžou vytvářet a spravovat žádosti o podporu v Azure Portal. Pro předplatitele Azure s [plánem podpory Azure](https://azure.microsoft.com/support/plans/) , který je přidružený k předplatnému, je k dispozici podpora pro účely jednoho vývoje pro sledování textu, sada senzorů, sada Speech SDK nebo Azure Cognitive Services.

  - Máte k vašemu předplatnému Azure přidružený [plán podpory Azure](https://azure.microsoft.com/support/plans/) ?  [Přihlaste se k Azure Portal](https://ms.portal.azure.com/) pro odeslání incidentu.
  - Potřebujete předplatné Azure? [Možnosti předplatného Azure](https://azure.microsoft.com/pricing/purchase-options/) vám poskytnou další informace o různých možnostech.
  - Potřebujete plán podpory? [Vybrat plán podpory](https://azure.microsoft.com/support/plans/)

### <a name="azure-kinect-on-premises-or-other-cloud-services"></a>Místní nebo jiné cloudové služby Azure Kinect

Technickou podporu s použitím sady SDK pro sadu senzorů a místní sady SDK pro sledování textu získáte pomocí lístku pro profesionální podporu na [portálu podpory společnosti Microsoft](https://support.microsoft.com/supportforbusiness/productselection?sapId=c49ea5bb-2b09-8612-be35-d55159732667).

### <a name="azure-kinect-dk-device"></a>Zařízení Azure Kinect DK

Než se obrátíte na hardwarovou podporu, ujistěte se, že jste nastavili a aktualizovali Azure Kinect DK. Pokud chcete otestovat, jestli zařízení funguje, použijte [prohlížeč Azure Kinect](azure-kinect-viewer.md). Další informace najdete na naší stránce s [technickou pomocí Azure Kinect DK](./index.yml) .
Můžete taky chtít zjistit naše [známé problémy a řešit](troubleshooting.md)potíže.

[Získejte pomoc](https://support.microsoft.com/supportforbusiness/productselection?sapId=f77b1b95-721e-43a0-2db8-b01e81a3f813) s funkcemi zařízení nebo senzorů, aktualizacemi firmwaru nebo možnostmi nákupu.

Další informace o nabídkách podpory najdete v podrobnějších informacích o [podpoře Microsoftu pro firmy](https://support.microsoft.com/help/4341255/support-for-business).

[Tady](https://www.microsoft.com/en-us/eu-declarations-compliance?activetab=pivot1:primaryr3) jsou deklarace shody EU pro hardwarové produkty Microsoftu.
