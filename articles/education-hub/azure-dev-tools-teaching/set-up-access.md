# <a name="setting-up-access-for-azure-dev-tools"></a>Nastavení přístupu pro vývojářské nástroje Azure

Existují dva způsoby, jak získat přístup k předplatnému, abyste mohli nasadit software pro studenty a obsazovat své laboratoře:
1. Stažením softwaru a klíčů z portálu Visual Studio Subscription.
1. Udělení přístupu studentům do obchodu Centrum Education Store.

Další informace najdete na [webu Azure for Education](https://azureforeducation.microsoft.com).

## <a name="for-administrators-only"></a>Jenom pro správce  
Správci přistupují k vývojářským nástrojům Azure prostřednictvím [portálu Visual Studio Subscription](https://my.visualstudio.com/).

Pokud chcete požádat o přístup k vašemu Visual Studio Subscription, můžete to provést prostřednictvím [portál pro správu](https://azureforeducation.microsoft.com/account/Subscriptions). Po schválení budete mít přístup k portálu předplatných sady Visual Studio. Pokud k portálu Visual Studio Subscription přistupujete poprvé, budete vyzváni k propojení účtu Microsoft s e-mailovou adresou, kterou jste zadali při registraci. K propojení těchto účtů budete potřebovat své ID předplatitele nebo číslo přístupu k výhodám, jméno a příjmení správce a e-mail uvedený během registrace. Po propojování účtů budete mít přístup k portálu pro předplatitele sady Visual Studio.

## <a name="for-students-faculty-and-administrators"></a>Pro studenty, vyučující a správce
Studences přístup k vývojářským nástrojům Azure prostřednictvím [centrum Education](https://aka.ms/devtoolsforteaching).

Studenti a vyučující můžou získat přístup ke všem výhodám stažení softwaru prostřednictvím Centrum Education. Centrum Education je sestaven v rámci Azure Portal a poskytuje studentům snadný přístup k celému katalogu softwaru a také k přístupu k celému katalogu [Microsoft Learn](/learn/) .
