- **[Monitorování pomocí Azure Data Studio řídicích panelů](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> Azure Data Studio je databázový Nástroj pro různé platformy pro odborníky na data, který využívá rodinu místních a cloudových datových platforem v systémech Windows, MacOS a Linux. Azure Data Studio nabízí moderní prostředí pro editory pomocí technologie IntelliSense, fragmenty kódu, integraci správy zdrojového kódu a integrovaný terminál. Je navržena s ohledem na uživatele datové platformy s integrovanými tabulkami sad výsledků dotazu a přizpůsobitelnými řídicími panely.

**[Monitorování pomocí Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana je webová aplikace pro open source analýzu a interaktivní vizualizace pro více platforem. Poskytuje grafy, grafy a výstrahy pro web při připojení k podporovaným zdrojům dat. Dá se rozšířit prostřednictvím systému modulů plug-in. Koncoví uživatelé můžou vytvářet komplexní řídicí panely monitorování pomocí interaktivního dotazu.

**[Log Analytics s Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana je otevřený řídicí panel vizualizace zdrojových dat pro Elasticsearch. Poskytuje možnosti vizualizace nad obsahem indexovaným v clusteru Elasticsearch. Uživatelé mohou vytvářet pruhové, spojnicové a bodové pruhy nebo výsečové grafy a mapovat je na velké objemy dat.
