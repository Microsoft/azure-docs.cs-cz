# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Azure Storage partneři pro primární a sekundární úložiště

Tento článek popisuje partnerské společnosti Microsoftu, které dodávají řešení NAS (Network Attached Storage) nebo síť SAN (Storage Area Network). Řešení může být místní, v Azure nebo hybridní řešení, které používá Azure Storage jako nákladově efektivní úroveň. Tato řešení můžou zákazníkům umožnit, aby v jakémkoli prostředí používali stejné řešení.
