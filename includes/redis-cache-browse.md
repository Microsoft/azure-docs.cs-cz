Pokud jste mezipaměť na řídicí panel nepřipnuli, najdete ji na webu [Azure Portal](https://portal.azure.com) po kliknutí na **Všechny služby**.

![Okno procházení Azure cache pro Redis](media/redis-cache-browse/redis-cache-browse.png)

Pokud chcete zobrazit mezipaměti, klikněte na **všechny služby** a vyhledejte **Azure cache pro Redis**. 

Pokud chcete zobrazit a nakonfigurovat nastavení určité mezipaměti, vyberte ji.

![Seznam mezipaměti Azure cache pro procházení Redis](media/redis-cache-browse/redis-caches.png)

Mezipaměť můžete zobrazit a nakonfigurovat z okna **Azure cache pro Redis** .

![Mezipaměť Azure pro Redis všechna nastavení](media/redis-cache-browse/redis-cache-blade.png)
