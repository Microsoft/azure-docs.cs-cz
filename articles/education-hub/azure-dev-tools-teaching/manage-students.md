# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Registrace studentů ve vývojových nástrojích Azure pro vzdělávání
Po zakoupení vývojářských nástrojů Azure pro vzdělávání budou mít všichni studenti, kteří se nacházejí v doméně vaší instituce, přístup k této nabídce automaticky. Během procesu nákupu budete požádáni o zadání e-mailové adresy správce předplatného. Tato doména ověřená na univerzitě bude odemknout studenty se stejnou e-mailovou doménou a udělí jim možnost přihlásit se a uplatnit výhody stahování softwaru. Pokud studenti ještě nemají účet Microsoft (MSA) nebo jsou součástí služby University Active Directory, zobrazí se jim výzva k jeho vytvoření, která odpovídá jejich e-mailové škole. Pokud studenti mají problémy s přístupem k Centrum Education, podívejte se na naši příručku pro řešení potíží níže nebo kontaktujte nás na adrese [https://aka.ms/studentsupport](https://aka.ms/studentsupport) .

## <a name="managing-access-for-students"></a>Správa přístupu pro studenty
Přístup pro studenty můžete spravovat na základě domény, kterou používají pro přihlášení k vývojářským nástrojům Azure pro vyučovací účely. Přečtěte si [portál pro správu](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) pro vývojářské nástroje Azure pro vzdělávání. Tam uvidíte všechna Vaše předplatná. Vyberte předplatné, pro které chcete spravovat přístup.
