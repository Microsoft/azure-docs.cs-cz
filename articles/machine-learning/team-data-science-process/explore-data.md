# <a name="explore-data-in-the-team-data-science-process"></a>Zkoumání dat v rámci vědeckého zkoumání týmových dat

Zkoumání dat je krok v rámci [vědeckého procesu týmového zpracování dat](overview.md).

Následující články popisují, jak prozkoumat data ve třech různých prostředích úložiště, která se obvykle používají v procesu datové vědy:

* Prozkoumejte data [kontejneru Azure Blob](explore-data-blob.md) pomocí balíčku Pythonu [PANDAS](https://pandas.pydata.org/) .
* Prozkoumejte [SQL Server](explore-data-sql-server.md) data pomocí SQL a pomocí programovacího jazyka, jako je Python.
* Prozkoumejte data [tabulky podregistru](explore-data-hive-tables.md) pomocí dotazů na podregistry.

Kromě toho video, [předzpracování dat v Azure Machine Learning Studio](https://azure.microsoft.com/documentation/videos/preprocessing-data-in-azure-ml-studio/), popisuje běžně používané moduly pro čištění a transformaci dat.
