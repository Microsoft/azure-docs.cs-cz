# <a name="assign-user-roles-and-permissions"></a>Přiřazování uživatelských rolí a oprávnění

**Příslušné role**

- Vlastník
- Manažer

Aby bylo možné získat přístup k funkcím souvisejícím s Marketplace nebo vaším vývojářským účtem, potřebujete mít k sobě přiřazenou jednu nebo více následujících rolí. Každá role má konkrétní sadu oprávnění k provádění určitých funkcí v rámci účtu.

| Role | Popis |
| ------------ | ------------- |
| Vlastník | Při vytvoření účtu vydavatele se partnerovi přiřadí role vlastnictví.<br><br>Podrobnosti oprávnění:<br><br>&#10004;&#160;&#160;vyplnit vlastnictví účtu.<br>&#10004;&#160;&#160;přidat typ uživatelů<br>&#10004;&#160;&#160;může doplňovat informace o vydaných a daních<br>&#10004;&#160;&#160;zobrazení sestav a souhrnů zpětné vazby<br>&#10004;&#160;&#160;zaregistrovat účet vydavatele do všech ostatních vývojářských programů |
| Manažer | &#10004;&#160;&#160;Správa uživatelů, rolí a oprávnění účtu vydavatele<br>&#10004;&#160;&#160;spravovat klienty<br>&#x2718;&#160;&#160;přístup k nastavení daní a výběrů |
| Vývojář | &#10004;&#160;&#160;nahrát balíček<br>&#10004;&#160;&#160;odesílat aplikace a doplňky<br>&#10004;&#160;&#160;zobrazení sestav využití pro podrobnosti telemetrie<br>&#x2718;&#160;&#160;přístup k finančním informacím nebo nastavení účtu |
| Obchodní Přispěvatel | &#10004;&#160;&#160;přístup k finančním informacím<br>Podrobnosti o cenách &#10004;&#160;&#160;sady<br>&#x2718;&#160;&#160;vytváření a odesílání nových aplikací a doplňků |

> [!NOTE]
> V případě programu komerčního obchodu se nepoužívají globální správce, obchodní přispěvatel, finanční přispěvatel a role na trhu. Přiřazení těchto rolí uživatelům se nijak neprojeví. Oprávnění uživatelům přidělují jenom role správce a vývojář.

Další informace o správě rolí a oprávnění v jiných oblastech partnerského centra, jako je Azure Active Directory (AD), zprostředkovatel Cloud Solution Provider (CSP), ovládací panel dodavatel (CPV), uživatelé typu Host nebo Microsoft Partner Network (MPN), najdete v tématu [přiřazení rolí uživatelů a oprávnění v partnerském centru](/partner-center/permissions-overview).

> [!NOTE]
> Každá Správa uživatelů, aktivity přiřazení rolí prováděné na těchto řádcích, budou v kontextu účtu, na kterém se nacházíte. Pokud potřebujete spravovat jiný účet, přečtěte si část o přepínání mezi prodejcem.
