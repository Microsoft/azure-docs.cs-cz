# <a name="help-and-support-options-for-azure-communication-services"></a>Možnosti pomoci a podpory pro komunikační služby Azure

Pokud máte problém nebo dotaz týkající se komunikačních služeb Azure, máte spoustu možností.

## <a name="create-an-azure-support-request"></a>Vytvoření žádosti o podporu Azure
S Azure máte k dispozici spoustu [možností podpory a plánů](https://azure.microsoft.com/support/plans/) , které můžete prozkoumat a zkontrolovat. Lístek podpory můžete vytvořit v [Azure Portal](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

## <a name="post-a-question-to-microsoft-qa"></a>Odeslat otázku do Microsoft Q&A

V případě rychlých a spolehlivých odpovědí na produkt nebo technické otázky, které byste mohli mít k dispozici v souvislosti s komunikačními službami Azure od Microsoft Engineers, Azure nejvíc MVP (MVP) nebo naší komunity, spolupracujte s námi na [Microsoft Q&A](https://docs.microsoft.com/answers/products/azure). 

Pokud nemůžete najít odpověď na váš problém hledáním, můžete odeslat novou otázku do složky Microsoft Q&A. Při vytváření otázky nezapomeňte použít [značku služby Azure Communication Services](https://docs.microsoft.com/answers/topics/azure-communication-services.html).

## <a name="post-a-question-on-stack-overflow"></a>Odeslat dotaz na Stack Overflow

Můžete se také pokusit o dotaz na Stack Overflow, který má velkou komunitu vývojářů a ekosystém. Komunikační služby Azure obsahují [vyhrazenou značku](https://stackoverflow.com/questions/tagged/azure-communication-services) .
