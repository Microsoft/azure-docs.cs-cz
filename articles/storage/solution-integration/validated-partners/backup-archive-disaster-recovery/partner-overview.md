# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Azure Storage archivů, zálohování a partnerů pro zotavení po havárii

V tomto článku se zvýrazňují partneři Microsoftu, které jsou integrované s Azure Storage pro archivaci a BCDR úlohy. Využijí výhod škálování a [nákladových výhod](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) Azure Storage k efektivnímu řešení problémů se zálohováním, vytvoření lokality pro zotavení po havárii nebo archiv nevyužitého obsahu pro dlouhodobé uchovávání. Díky všem standardům dodržování předpisů, které Azure Storage splňuje, a s [neproměnlivým řízením úložiště](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) a životního cyklu můžou tato řešení snadno nahradit zálohy založené na pásce a nabízet úsporný Server pro úsporu na vyžádání.
