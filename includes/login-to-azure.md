## <a name="sign-in-to-azure"></a>Přihlásit se k Azure

Pomocí Azure CLI vytvoříte prostředky potřebné k hostování vaší aplikace v Azure. Pokud spouštíte příkazy Azure CLI ve službě Cloud Shell, jste již přihlášeni. Pokud chcete příkazy Azure CLI spouštět místně, přihlaste se ke svému předplatnému Azure pomocí příkazu [az login](/cli/azure/) a postupujte podle pokynů na obrazovce.

```azurecli
az login
```
