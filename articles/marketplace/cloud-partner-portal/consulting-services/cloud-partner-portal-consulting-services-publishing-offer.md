# <a name="azure-and-dynamics-365-consulting-service-offer"></a>Nabídky Azure a Dynamics 365 konzultační služby

Tento článek vysvětluje, jak publikovat nabídku na konzultační služby buď Microsoft <a href="https://azuremarketplace.microsoft.com">Azure Marketplace</a> nebo <a href="https://appsource.microsoft.com">AppSource</a>. Řešení založené na službě Microsoft <a href="https://dynamics.microsoft.com">Dynamics 365</a>, <a href="https://powerbi.microsoft.com">Power BI</a>, a <a href="https://powerapps.microsoft.com">PowerApps</a> mohou být uvedeny v AppSource. Jiné nabídky založené na jiné službě Microsoft <a href="https://azure.microsoft.com/services">služeb Azure</a> výpis je možný v Tržišti Azure Marketplace.

## <a name="publishing-benefits"></a>Publikování výhody

Publikování do některé z Tržiště Microsoftu má následující výhody:

- Propagace vaší společnosti využitím značky společnosti Microsoft.
- Potenciálně oslovit uživatele po více než 100 milionů operací služeb Office 365 a Dynamics 365 na AppSource a Oslovte víc než 200 000 organizací prostřednictvím Azure Marketplace.
- Získejte vysoce kvalitní zájemce z těchto tržišť s řešeními.
- Získejte služby povýšen pomocí Microsoft teams pole a prodej po telefonu.

## <a name="publish-a-consulting-service-offer"></a>Publikovat nabídku na konzultační služby

Tato část popisuje, jak prvky publikováním nabídky na konzultační služby. Publikování je rozdělena na následující hlavní oddíly:

- [Konzultační služby požadavky](./cpp-consulting-service-prerequisites.md) požadavky na publikování konzultační služby nabízejí pro seznamy:
 
    - Dynamics 365 for Customer Engagement 
    - Dynamics 365 pro Finance a provoz 
    - Dynamics 365 Business Central 
    - Power BI 
    - PowerApps
- [Vytvořit novou nabídku](./cpp-consulting-service-create-offer.md) seznam kroků potřebných k vytvoření konzultační položku nabídky služby přes portál partnerů cloudu:
    - [Nabídka nastavení](./cpp-consulting-service-define-offer-settings.md).
    - [Zadejte podrobnosti prezentace a jestli se mají publikovat na webu Azure Marketplace nebo v AppSource.](./cpp-consulting-service-storefront-details.md)
- [Publikování nabídky](./cpp-consulting-service-publish-offer.md) vysvětluje, jak odeslat nabídku k publikování.

## <a name="next-steps"></a>Další postup

Před provedením těchto kroků k publikování vaší nabídky konzultační služby, musí splňovat [požadavky](./cpp-consulting-service-prerequisites.md) pro publikování nabídky na konzultační služby.
