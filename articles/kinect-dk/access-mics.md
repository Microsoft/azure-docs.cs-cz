# <a name="access-azure-kinect-dk-microphone-input-data"></a>Přístup ke vstupním datům mikrofonu Azure Kinect DK

[Sada Speech SDK pro rychlý Start](../cognitive-services/speech-service/index.yml) poskytuje příklady použití pole mikrofonu Azure Kinect DK v různých programovacích jazycích.
Podívejte se například na rozpoznávání **řeči v C++ ve Windows pomocí sady Speech SDK pro** rychlý Start. Kód je k dispozici [z GitHubu](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp).

Přístup k poli mikrofonu také prostřednictvím rozhraní API systému Windows. Podrobnosti o dokumentaci k systému Windows najdete v následujících dokumentech:

* [Architektura zvuku Windows](/windows-hardware/drivers/audio/windows-audio-architecture)
* [Dokumentace k Windows. Media. Capture](/uwp/api/Windows.Media.Capture)
* [Kurz pro zachycení webové kamery](/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [Informace o zvuku USB](/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

Můžete také zkontrolovat [specifikace hardwaru pro pole mikrofonu](hardware-specification.md#microphone-array).
