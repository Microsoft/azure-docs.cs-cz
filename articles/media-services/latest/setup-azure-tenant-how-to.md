# <a name="how-to-find-your-azure-tenant-id"></a>Jak najít ID tenanta Azure

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Najít ID tenanta na portálu

1. Přihlaste se na [Azure Portal](https://portal.azure.com).
1. Vyberte ikonu Azure Active Directory v části služby Azure. Otevře se stránka Active Directory.
1. Vaše ID tenanta najdete v části **základní informace**.
1. Zkopírujte ID tenanta a vložte ho do textového dokumentu podle vlastního výběru pro pozdější použití.
