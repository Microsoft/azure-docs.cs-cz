# <a name="microsoft-security-response-center"></a>Středisko Microsoft Security Response Center

Je Microsoft Security Response Center (MSRC) vedené z celého světa nejzkušenějších odborníci na zabezpečení. Jejich identifikovat, monitorování, odpovědět a řešit bezpečnostní incidenty a místní a cloudové ohrožení zabezpečení hodin každý den v roce.

Kromě její průběžné práce týmu MSRC má celou řadu materiálů, které jsou k dispozici, tak, aby můžete pochopit, jak zabezpečit prostředky Azure a nasazení efektivněji.

## <a name="the-msrc-blog"></a>Blog střediska MSRC

[Blog střediska MSRC](https://blogs.technet.microsoft.com/msrc/) je místem, kde můžete získat nejnovější informace týkající MSRC stavu ochrany před hrozbami v cloudu.

## <a name="white-papers"></a>Dokumenty white paper

MSRC publikoval celou řadou [dokumenty white paper](https://technet.microsoft.com/library/bb969102.aspx) , které pomůžou pochopit, co dělají a jak dělat. Některé poskytují přehled o zabezpečíme, jak v cloudu společnosti Microsoft a také užitečné informace o tom, jak můžete použít stejnou konfiguraci zabezpečení.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Zabezpečení výzkumník zapojení a hojnosti programy

MSRC podporuje spolupráci a vztahy s výzkumníci globálně pro přechod zabezpečení produktu společnosti Microsoft.

Programy společnosti Microsoft hojnosti platit výzkumní pracovníci pro:

- Využívání nové techniky
- Obranné nápady, která snižuje riziko zneužití nové
- Identifikace kritické chyby zabezpečení ve službě Microsoft místních a cloudových softwaru.

## <a name="next-steps"></a>Další postup

Další informace o těchto programů na [MSRC chyb hojnosti](https://technet.microsoft.com/security/dn425036) stránky a [blog střediska MSRC](https://blogs.technet.microsoft.com/msrc/).

Další informace o MSRC, navštivte prosím [domovské stránce střediska MSRC](https://technet.microsoft.com/library/dn440717.aspx).
