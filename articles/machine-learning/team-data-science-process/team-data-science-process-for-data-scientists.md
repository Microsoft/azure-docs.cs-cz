# <a name="team-data-science-process-for-data-scientists"></a>Vědecké zpracování týmových dat pro odborníky přes data

Tento článek poskytuje pokyny pro sadu cílů, které se obvykle používají k implementaci komplexních řešení pro datové vědy s technologiemi Azure. Provede vás:

- Princip úlohy analýzy
- používání vědeckého zpracování týmových dat
- použití Azure Machine Learning
- Základy přenosu dat a úložiště
- poskytování dokumentace ke zdroji dat
- používání nástrojů pro zpracování analýz

Tyto školicí materiály se týkají TDSP (Team data Sciences) a softwaru a nástrojů společnosti Microsoft a open source, které jsou užitečné pro plánování, spouštění a doručování řešení pro datové vědy.
