#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Instalace adaptéru StorSimple pro SharePoint
1. Zkopírujte instalační program na server webového front-endu (WFE), který je také nakonfigurován pro spuštění webové aplikace centrální správy služby SharePoint. 
2. K přihlášení k serveru WFE použijte účet s oprávněními správce.
3. Dvakrát klikněte na instalační program. Spustí se Průvodce instalací adaptéru StorSimple pro službu SharePoint. Kliknutím na tlačítko **Další** zahajte instalaci.
   
    ![Úvodní stránka Instalace adaptéru StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. Na stránce konfigurace instalace adaptéru StorSimple pro SharePoint vyberte umístění instalace, zadejte IP adresu síťového rozhraní DATA 0 na zařízení StorSimple a pak klikněte na **Další**. 
   
    ![Stránka konfigurace instalace adaptéru StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Na stránce potvrzení instalace klikněte na **nainstalovat**.
   
    ![Stránka pro potvrzení nastavení adaptéru StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kliknutím na tlačítko **Dokončit** zavřete Průvodce instalací nástroje.
   
    ![Stránka dokončení nastavení adaptéru StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Otevřete stránku centrální správy služby SharePoint. Měla by se zobrazit skupina konfigurace StorSimple, která obsahuje adaptér StorSimple pro SharePointové odkazy.
8. Přejít k dalšímu kroku: [Konfigurace RBS](#configure-rbs).
