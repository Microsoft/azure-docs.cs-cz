# <a name="overview-of-the-iso-27001-blueprint-sample"></a>Přehled ukázky podrobného plánu ISO 27001

Ukázka podrobného plánu ISO 27001 poskytuje ochranné mantinely zásad správného řízení s využitím [Azure Policy](../../../policy/overview.md), které pomáhají vyhodnotit specifické kontroly ISO 27001. Tento podrobný plán pomáhá zákazníkům nasadit základní sadu zásad pro libovolnou architekturu nasazenou v Azure, která musí implementovat kontroly ISO 27001. K dispozici jsou dva další podrobné plány ISO 27001, které vám pomohou nasadit [základní architekturu](../iso27001-shared/index.md) a [úlohu ASE/SQL](../iso27001-ase-sql-workload/index.md).

## <a name="control-mapping"></a>Mapování kontrol

Oddíl mapování kontrol poskytuje podrobné informace o zásadách zahrnutých v tomto podrobném plánu a o tom, jak tyto zásady řeší různé kontroly v rámci standardu ISO 27001. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje případné nedodržení přiřazených zásad. Další informace najdete v tématu [Azure Policy](../../../policy/overview.md).
