# <a name="supported-browsers"></a>Podporované prohlížeče 

Azure dosah podporuje následující prohlížeče. Doporučujeme používat nejnovější prohlížeč, který je kompatibilní s vaším operačním systémem. 

* Microsoft Edge (nejnovější verze)
* Safari (nejnovější verze, jen Mac)
* Chrome (nejnovější verze)
* Firefox (nejnovější verze)

## <a name="chrome-incognito-mode"></a>Režim anonymním pro Chrome

 Aby dosah Studio fungovalo, musí být pro anonymním pro Chrome zakázané blokování souborů cookie třetích stran zakázané.

## <a name="chromium-edge-inprivate-mode"></a>Režim InPrivate Edge pro chrom

Aby aplikace dosah Studio fungovala, musí být zakázána funkce InPrivate Edge na základě striktního sledování.
