> [!TIP]
> Obsah tohoto článku se týká původní služby Azure Table Storage. Teď ale existuje alternativní nabídka pro úložiště tabulek: Azure Cosmos DB rozhraní API pro tabulky. Toto rozhraní API nabízí vyšší výkon a dostupnost, globální distribuci a automatické sekundární indexy. Je také k dispozici v režimu bez [serveru](../articles/cosmos-db/serverless.md) s využitím spotřeby. Mezi rozhraní API pro tabulky v Azure Cosmos DB a úložišti tabulek Azure je několik [rozdílných funkcí](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) . Další informace najdete v tématu [Azure Cosmos DB rozhraní API pro tabulky](../articles/cosmos-db/table-introduction.md).
