<a name="what-is-test-drive"></a>Co je to Test Drive?
===================

Testovací verze je skvělým způsobem, aby ukazovala nabídky pro potenciální zákazníky tím, že je možnost \'před zakoupením vyzkoušet\'výsledkem zvýšení převodu a vytváření vysoce kvalifikovaných prodejních příležitostí.

Po zadání kontaktní údaje najdete, zákazníkům přístup k prostředí předem připravené testovací verze: praktické, samostatně na základě zkušební verzi produktu\'s klíčových funkcích a výhodách je ukázáno v reálném scénáři implementace.

Test Drive umožňuje přineste svůj produkt do života a generování vysoce kvalifikovaných prodejních příležitostí v procesu.

<a name="how-does-a-test-drive-work"></a>Jak funguje Test Drive?
---------------------------

<a name="what-goes-on-in-the-background"></a>Co pokračuje na pozadí?
-------------------------------

Služba Test Drive je vytvořená průběžné podpory a sloužit vašim zákazníkům bez nutnosti jakékoli ruční práce od vás. Jako vydavatel vaše úloha není pro správu a konfiguraci nastavení testu jednotky z theCloud portál pro partnery, a pak toto nastavení přímo bude k dispozici pro vaše zákazníky.

Důvodem je, po nastavení konfigurace pro vaše testovací verze, bude každý Test Drive managed instance, která se nasadí na vyžádání pro zákazníka o to požádá. Po přiřazení instance Test Drive Test Drive je k dispozici pro použití pro nastavené doby nenaváže a pak se odstraní vytvořit místo pro další odběratele.
